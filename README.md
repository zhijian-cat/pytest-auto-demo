## 📖 项目简介

本项目是一个基于 **Pytest + Requests** 的接口自动化测试框架示例，用于学习和实践：

- ✅ Pytest测试用例编写（fixture、参数化、mark标签）
- ✅ Mock模拟外部依赖（unittest.mock）
- ✅ Allure测试报告生成
- ✅ GitHub Actions CI持续集成

## 🛠️ 技术栈

| 工具 | 版本 | 用途 |
| --- | --- | --- |
| Python | 3.10+ | 编程语言 |
| Pytest | 7.x | 测试框架 |
| Requests | 2.x | HTTP请求 |
| Allure | 2.x | 测试报告 |
| GitHub Actions | — | CI/CD流水线 |
